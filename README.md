## Installation

In the project directory run:

### 'yarn'

## Development mode

In the project directory run:

### `yarn dev`

## Production mode

In the project directory run:

### `yarn build`

After that you'll get the "build" folder. To run server with the application run:

### `yarn start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
